# Image Guidelines for Flow Studios

## Folder Structure

```
public/images/
├── modals/          # Modal popup images (newsletter, etc.)
├── products/        # Product catalog images
├── hero/            # Hero section banners
└── brand/           # Brand logos and assets
```

## Image Specifications

### Newsletter Modal Image
- **Recommended Size**: 800x800px (1:1 aspect ratio)
- **File Format**: JPG or WebP
- **File Name**: `newsletter-hero.jpg`
- **Location**: `public/images/modals/newsletter-hero.jpg`
- **Description**: Calming yoga/movement image to complement the newsletter signup

### Product Images
- **Recommended Size**: 1200x1500px (4:5 aspect ratio)
- **File Format**: JPG or WebP
- **Location**: `public/images/products/`

### Hero Banners
- **Recommended Size**: 1920x800px (desktop), 800x1000px (mobile)
- **File Format**: JPG or WebP
- **Location**: `public/images/hero/`

### Brand Assets
- **Logo**: SVG preferred (scalable)
- **Location**: `public/images/brand/`

## Best Practices
- Use WebP format for better compression when possible
- Always provide 2x resolution for retina displays
- Optimize images before uploading (use tools like TinyPNG)
- Use descriptive file names (e.g., `newsletter-hero.jpg` not `img1.jpg`)
